# Common

The `common` package contains reusable components shared across all Genie packages.

## Responsibilities

- Shared data models
- Configuration management
- Logging
- Utility functions
- Base interfaces and abstract classes
- Shared prompts
- LLM abstractions
- Tool definitions

## Guidelines

Only place code here if it is expected to be reused by two or more packages.

Avoid placing business logic in `common`.
